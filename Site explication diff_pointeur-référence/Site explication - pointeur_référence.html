Pointeur vs Référence en C++

Dans le langage C++, les pointeurs et les références permettent tous deux de travailler sur des variables indirectement, mais leur fonctionnement et leur utilisation sont différents. Voici une explication simple avec exemples.

1. Le pointeur

Un pointeur est une variable qui contient l’adresse mémoire d’une autre variable. Il doit être déréférencé (avec l'ajout de "*") pour accéder à la valeur. Il en va de soit que deux pointeurs peuvent pointer vers la même variable ou vers des variables différentes.

Exemple :


    int x = 10;
    int* p = &x;  // p contient l'adresse de x

    cout << *p << endl;  // Affiche la valeur de x (10)

    cout << p << endl;   // Affiche l'adresse de x (0xxxxxxxx)

    *p = 20;     // On modifie x via le pointeur

    cout << x << endl;  // Affiche la nouvelle valeur de x (20)

Après exécution, la variable x vaut 20.

2. La référence

Une référence est un alias donné à une variable existante, un nom alternatif (un surnom de cette dernière). Elle n’est pas stockée comme une variable différente. Elle doit être initialisée dès sa création et ne peut pas changer de cible. C'est aussi possible d'avoir plusieurs références pointant vers la même variable.

Exemple :


    int x = 10;
    int& r = x;   // r est une référence à x

    cout << r << endl;   // Affiche la valeur de x (10)

    r = 30;       // On modifie directement x

    cout << x << endl;   // Affiche la nouvelle valeur de x (30)

Après exécution, la variable x vaut 30.

3. Tableau de comparaison

Critère Pointeur Référence

Peut changer de cible Oui Non
Déclaration int* p int& r
Accès à la valeur Il faut déréférencer (*p = nouvelle_valeur) Directement (r = nouvelle_valeur)
Unicité Non Non
Représentation en mémoire Variable séparée contenant une adresse Alias de la variable d’origine

4. Exemple complet : avant / après

Avec pointeur :


    int x = 5;
    int* p = &x;
    *p = 40;   // x devient 40

Avec référence :


    int x = 5;
    int& r = x;
    r = 40;    // x devient aussi 40

Conclusion

En résumé :

  • Le pointeur stocke une adresse et peut être nul ou changer de cible.
  • La référence est un alias direct, toujours valide et permanent.
  • Les deux permettent de modifier une variable indirectement.
